=================================================================
    INSTRUCCIONES DE INSTALACIÓN - EasyFind Suite
=================================================================

REQUISITOS PREVIOS:
-------------------
1. Windows 10 o superior
2. Python 3.10 o superior instalado
3. Conexión a Internet

PASOS DE INSTALACIÓN:
---------------------

1. INSTALAR PYTHON (si no está instalado)
   - Descargar desde: https://www.python.org/downloads/
   - Durante la instalación, MARCAR la opción:
     ☑ "Add Python to PATH"

2. VERIFICAR INSTALACIÓN DE PYTHON
   - Abrir CMD (Símbolo del sistema)
   - Ejecutar: python --version
   - Debe mostrar: Python 3.10.x o superior

3. INSTALAR LIBRERÍAS NECESARIAS
   - Abrir CMD en la carpeta donde descomprimiste el ZIP
   - Ejecutar: pip install -r requirements.txt
   - Esperar a que termine la instalación (puede tardar varios minutos)

4. EJECUTAR LA APLICACIÓN
   - Doble clic en: EasyFind_Suite.exe
   - La interfaz gráfica debe abrirse

FUNCIONES PRINCIPALES:
----------------------

📥 ACTUALIZAR BD:
   - Ejecuta los 12 bots recolectores en paralelo
   - Actualiza las bases de datos de todas las tiendas
   - Puede tardar 30-60 minutos dependiendo de la conexión
   - Muestra progreso en tiempo real de cada bot

🔎 BUSCAR PRECIOS:
   - Busca precios de productos en PRODUCTOS.xlsx
   - Genera archivo Resultado.xlsx con precios comparados
   - Requiere que las bases de datos estén actualizadas

🛑 DETENER:
   - Detiene el proceso en curso
   - Guarda los datos procesados hasta el momento

SOLUCIÓN DE PROBLEMAS:
----------------------

❌ Error: "python no se reconoce como comando"
   → Reinstalar Python marcando "Add Python to PATH"
   → O buscar donde está instalado Python y agregarlo manualmente al PATH

❌ Error: "No module named 'requests'" (o similar)
   → Ejecutar: pip install -r requirements.txt
   → Si persiste: pip install pandas playwright beautifulsoup4 openpyxl requests

❌ "Actualizar BD" se completa instantáneamente sin progreso
   → Verificar que la carpeta Bots_recolectores/ existe y tiene 12 archivos .py
   → Verificar que Python está instalado correctamente
   → Verificar que las librerías están instaladas: pip list

❌ "Buscar Precios" no funciona o da error
   → Verificar que la carpeta browsers/ existe y pesa ~640 MB
   → Verificar que PRODUCTOS.xlsx existe en la misma carpeta del ejecutable
   → Verificar que la carpeta TIENDAS/ tiene archivos .xlsx

❌ Error: "Falta archivo PRODUCTOS.xlsx/csv"
   → Crear un archivo Excel llamado PRODUCTOS.xlsx
   → Debe tener una columna llamada "ItemName", "Descripcion" o "Producto"
   → Colocar los nombres de productos a buscar en esa columna

ESTRUCTURA DE ARCHIVOS NECESARIA:
----------------------------------

EasyFind_Suite.exe          ← Ejecutable principal
PRODUCTOS.xlsx              ← Archivo con productos a buscar (para Buscar Precios)
requirements.txt            ← Lista de dependencias Python
Bots_recolectores/          ← Carpeta con 12 scripts de bots (OBLIGATORIA)
TIENDAS/                    ← Carpeta con 12 bases de datos Excel (OBLIGATORIA)
browsers/                   ← Navegador Chromium para scraping (OBLIGATORIA, ~640 MB)

NOTAS IMPORTANTES:
------------------

⚠️ La primera vez que ejecutes "Actualizar BD" tardará bastante tiempo
   (30-60 minutos) porque descarga información de 12 tiendas diferentes.

⚠️ Se recomienda ejecutar "Actualizar BD" semanalmente para mantener
   las bases de datos actualizadas con los últimos productos.

⚠️ El archivo Resultado.xlsx se sobrescribe cada vez que ejecutas
   "Buscar Precios". Guarda copias si necesitas mantener resultados anteriores.

⚠️ Durante "Actualizar BD", el sistema previene que la PC entre en
   suspensión automáticamente. Esto es normal y se revierte al terminar.
